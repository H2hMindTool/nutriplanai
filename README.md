# 🥗 NutriPlanAI - Elite Nutrition SaaS

A state-of-the-art Nutriton SaaS platform featuring AI-driven meal planning, real-time evolution tracking, and industry-standard user experience.

## 💎 Premium Features

### 🚀 Advanced Navigation Ecosystem
- **Hybrid Model**: Universal Bottom Navigation + Global Top Bar.
- **Elite Sidebar**: Glassmorphic sliding menu for full app exploration.
- **Mobile First**: Optimized for seamless tracking and navigation on the go.

### 🤖 AI Support & Persistence
- **GPT-Style Chat**: Immersive support interface with brand persistence.
- **Smart History**: All AI interactions and meal plans are saved to Supabase for instant retrieval.
- **Context-Aware**: AI support that knows your goals and nutritional profile.

### 📈 Evolution Tracking
- **SVG Data Visualization**: Real-time weight and progress charts.
- **Mobile Fixed**: High-density SVG rendering for perfect visibility on all screens.
- **Simplified Input**: Quick weight logs with instant history updates.

## 🛠️ Technology Stack

- **Core**: [Next.js](https://nextjs.org/) (App Router)
- **Database & Auth**: [Supabase](https://supabase.com/)
- **AI Engine**: [OpenAI GPT-4o](https://openai.com/)
- **UI Architecture**: Glassmorphism & Vanilla CSS (Industry Standard)
- **Deployment**: Integrated CI/CD via GitHub.

## 🏁 Getting Started

1. **Clone & Install**:
   ```bash
   git clone https://github.com/H2hMindTool/nutriplanai.git
   npm install
   ```

2. **Environment Setup**:
   Configure `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, and `OPENAI_API_KEY` in `.env.local`.

3. **Launch**:
   ```bash
   npm run dev
   ```
